# Сервис управления рассылками (Mailing Service)

Веб-приложение на Django для создания и управления email-рассылками.

## Функциональность

- Управление клиентами (получателями рассылок)
- Управление сообщениями
- Управление рассылками (создание, редактирование, удаление)
- Автоматическое обновление статуса рассылок
- Отправка писем через Gmail/Yandex
- История попыток отправки
- Отчеты и статистика
- Разграничение прав доступа
- Отправка через интерфейс и командную строку

## Технологии

- Python 3.10+
- Django 5.0
- PostgreSQL / SQLite
- Bootstrap 5
